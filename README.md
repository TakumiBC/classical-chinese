# 前言

當今市面上的文言文教材，數目非常之多；如果要學習怎樣閱讀文言文，根本不是一件難事。但自民初的「新文化運動」以來，隨着中國傳統私塾教育的式微與白話文教育的推行，絕大多數中國人已經喪失了用文言文寫作的能力。

相信不少的讀者，都曾於中小學階段接受過文言文教育。但當今的文言文教育，事實上只是去除了傳統文言文教育「唸、背、打」的「打」這一步，而並未進行科學地改造。學生要讀古文、背古文、背註釋、背翻譯（因為多數學生的文言水平不足以支撐他們去自行翻譯），卻不再練習以文言文作文。於是文言文在課堂中，只剩下了閱讀。

至於給中文系專業的古代漢語教材，則僅適合有語言學基礎者；就算有語言學基礎者，讀着亦要感到費勁。試模擬他們的方式，對「今晨赴校」這句話做分析：

> 「今晨赴校」一語，乃第一人稱敘述語境下主語零形化之簡約句式。倘不省略主語，可還原為「吾今晨赴校」，惟因語篇主體已由敘述者預設，故表層結構中主位成分發生省略，形成無顯性主語之時間框架化動核結構。其中，「今晨」為時間指示語，於句法層面充任前置時間狀語，對後續謂詞性核心「赴校」進行時間域限定；於語用層面則標記事件發生之近指過去時段，兼具敘事起點設定功能。「赴」為趨向動詞，含有由敘述主體所在位置向目標處所移動之語義特徵；「校」為處所性名詞，於「赴」後形成目標補足成分。全句遂構成「時間狀語＋趨向動詞＋目標處所」之簡約敘事句型。

實在是難懂。

不少的志士，都曾因為文言寫作的斷層而感到惋惜，故想寫一本《文言寫作教材》出來。但他們也忽略了一個關鍵問題：傳統文言教育，除了讀、寫以外，還同時涵蓋了「聽」。

現代人讀古文的辦法，絕大多數都是「朗誦」。而古代私塾學生使用的方法，是「吟誦」：先生以代代相傳的、酷似歌曲般的一種音樂性腔調，將文章或經文讀出，再由學生反覆跟讀學習。這不正是語言教學中最重要的一環「聽」嗎？古人所說「熟讀唐詩三百首，不會作詩也會吟」，其中「吟」便是指吟誦。

這種吟誦，好處非常多。其一，在於音樂性的腔調使得背誦水到渠成，就像當今的小孩子背歌詞一樣，無需花許多功夫，且終身難忘。其二，吟誦「依字行腔」的特點，使人在剖析文義前，便能體會到文章的深層意義與感情色彩。

而我們都知道，無論在學習中文還是外語時，最完整、最有效的語文學習，往往不能只偏重閱讀，而應同時照顧聽、說、讀、寫四方面。即使西方在教授拉丁文這類「死語言」時，也常重視朗讀、口說、閱讀與作文。那麼文言文教育中，能否出現「說」呢？

答案是：可以。比如，中國語言學之父趙元任先生在美國教外國人文言文時，便直接在課堂上以文言文進行了大量口頭練習、課堂對話與寫作練習。[^1] 假如連外國人都可以做到，那我們中國人有什麼做不到的呢？他甚至說，「無論活語言還是死語言，要學得好，都應當當作活的來教」。這實在是語言教學的正確方法。

由此可知，今日文言文教育的根本問題，並不只是缺少一本寫作教材，而是缺少一套完整的語文教材。文言文應當是一種活的書面語，是同時可以輸入、輸出的。

國學大師錢賓四先生曾有感白話文教育使得學生古書閱讀能力下降，便撰寫了《無師自通中國文言自修讀本之編輯計劃書》[^2]，預備要編一本龐大的教科書，可惜此書終未完成。同時，此計劃之重心仍與現有眾多教材相近，只關注「讀」這一項。

國父　孫中山先生更曾指出：「以無文法之學，故不能率由捷徑，以達速成，此猶渡水之無津梁舟楫，必當繞百十倍之道路也。」又期望「吾國好學深思之士，廣搜各國最近文法之書，擇取精義，為一中國文法。」[^3] 　國父的這番話，實在是有道理。古時候學文言寫作，使用「唸、背、打」的方法，確實有效。但私塾乃是為了培養經學家。經學家有必要一字不差背完四書，但簡單的寫文言文不必要背完四書。我們學英文，也不見得要把《聯邦黨人文集》背完才能寫文章吧？　國父作為私塾教育的親歷者，想必比我們諸位更加清楚。

當今，分析文言語法的書目亦已極多，有如王力《古代漢語》與楊伯峻《文言文法》等。但問題在於，　國父所期盼的這本語法書，是要能指導初學者寫作的，而非只是單純供語言學家分析用。本人亦曾按照錢賓四先生未完成之計劃試編數課，愈發覺得目前文言教育最需要者，正是「寫」。不妨將二人之計畫合為一本教材，由淺入深，以聽、說帶動讀、寫，不但完成二位先生的意願，更能填補目前市面上文言寫作教材的空白，還能成為迄今為止最科學、最高效的文言文教材。

我以為，用一個不太嚴格符合語言學的定義來說：文言文既是母語，又是外語。說它是母語，是因為它用的是漢字，且絕大多數字詞乃至語法都與白話中文相通。說它是外語，是因為普通讀者雖不經特別訓練就能認出多數的字，卻未必能掌握文言與白話之間的微妙區別，更未必能寫作。

因此，本書同時借鑑了第二語言教學與母語語文教學之法，以聽讀、跟誦、對話為引導，先使讀者能流暢閱讀常見文言文章，並能寫出合乎語法的文言短文；再以屬對、文體訓練等方式，使讀者逐步精進。教材整體編排，為培養讀者崇基、尊孔、愛國的三大品德而服務。

本書主要為自學者而作。作者本人在文言寫作這一方面，亦是自學者，深感自學者滿懷熱情而難尋門道之痛。因此，本書力求由淺入深，詳細講解，使讀者即使無師亦能自通。不過，若能有教師指點，尤其是若能有教師批改作文，則進步必更為迅速。自學固然可以掌握，但若有明師指導，則能事半功倍，在任何學科的學習都是如此。

希望讀者自強不息，堅持把本書認真讀完。如此，便能手握文言文這一把最關鍵的鑰匙，從容踏進國學的大門。
