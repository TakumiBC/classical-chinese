# 第一單元：基本口語

本單元為全書的第一單元。依照聽、說、讀、寫之編寫精神，本書不依慣例，而由口語開始教授文言文。其根本目的，在於將文言文變成活語言，以及培養讀者之興趣。

文言文在日常口語中，已不再使用；即使古代學習文言文者，也不會用文言文與人進行口語溝通。但為將其變為活語言，本書參照趙元任先生教外國人文言文之做法，並依照教授第二語言的各種語言學研究，模擬先秦的口語場景，使讀者從第一課起便能聽、能說、能寫。

本單元的課文以短對話為主，依絕大多數二語教材的編寫精神，先介紹打招呼、自我介紹、告別等基本口語場景。隨後，再引入篇幅稍長、但仍以口語佔絕對比重的課文。

同時，本單元將包含罵人，尤其是會包含髒話的使用。梁實秋曾經說過：「古今中外沒有一個不罵人的人。」[^1] 絕大多數教材都會刻意迴避這些話題，然而這實在是口語交流中不可或缺的。不包含罵人的口語環境，絕不是完整的口語環境。小孩子學外語，通常最先學會的，也是罵人和髒話。與其迴避，不如大膽地引入，使讀者萌生興趣，也更能模擬活語言的真實場景。

## 准入標準

- 讀者會中文，且認識漢字。

## 單元結束後，讀者應能

- 以文言口語進行基本對話。
- 使用基本問候、自我介紹、告別、致謝、道歉等口語。
- 識別並演練基本責備語、詈語與髒話。
- 了解文言文簡潔、省略、重語氣、重場景的特點。

[^1]: 秋郎，《罵人的藝術》（上海：新月書店，1927），頁1。
